# 🛡️ Unified Cyber Fraud Intelligence Platform

> **AI-Driven Correlation of Cybersecurity Telemetry & Transactional Behaviour**

An AI-powered platform that correlates cybersecurity telemetry, threat intelligence, and transactional behaviour to detect, analyze, and explain cyber fraud in real time. Built using modern web technologies and Google's Gemini AI, the platform empowers financial institutions with intelligent fraud detection, explainable insights, and actionable security recommendations.

---

## 📖 Overview

As cyber fraud continues to evolve, traditional fraud detection systems struggle to identify sophisticated attacks due to isolated analysis of security events and transaction data.

The **Unified Cyber Fraud Intelligence Platform** addresses this challenge by combining:

- 🔐 Cybersecurity Telemetry
- 💳 Transaction Behaviour
- 🌐 Threat Intelligence
- 🤖 Explainable Artificial Intelligence (XAI)

into a unified intelligence platform that enables proactive fraud detection and investigation.

# 🚀 Workflow

1. User submits cyber fraud indicators such as:
   - IP Address
   - URL
   - Domain
   - Transaction Details
   - Suspicious Activity

2. The platform validates and preprocesses the input.

3. Google Gemini AI analyzes the submitted information.

4. Threat Intelligence correlates multiple security indicators.

5. Fraud Risk Score is generated.

6. Explainable AI produces a detailed investigation report.

7. Interactive dashboard displays:
   - Risk Analysis
   - Security Insights
   - Threat Indicators
   - Recommended Actions

# ⚡ Installation

Clone the repository

```bash
git clone https://github.com/aakashahuja912-cpu/unified-cyber-fraud-intelligence.git
```

Navigate to the project

```bash
cd unified-cyber-fraud-intelligence
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the development server

```bash
npm run dev
```
